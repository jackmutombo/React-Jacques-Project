In order for the front end to run you have to start your api application.
your API URL need to set up in the React app.
To set up the URl, open the src folder, then apis, then staffProductAPI.js and change the port number 
to the corresponding port number you api is running.
run the application click on register. Note that username must be equal to the email address. monitor the console if you getting the token logged. if successul click on log in to log in.
